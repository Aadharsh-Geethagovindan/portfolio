## Technical Details
- Next.js + React + Tailwind
- Supabase for content storage and user/admin access
- Deployed on Vercel

## Challenges & Solutions
<Callout title="Challenge">
The site needed to be maintainable by non-developers after handoff.
</Callout>

<Callout title="Solution">
Designed a content model in Supabase + admin editing tools so updates require no code changes. 
</Callout> 

<Callout title="Outcome">
Admins can update content and scheduling workflows directly through the site.
</Callout>
