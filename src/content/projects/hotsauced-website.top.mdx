## Overview
HotSauced is a team hub for a recreational sports group. The architecture is complete and supports content editing by team admins; the remaining work is content finalization with the leadership group.

## Key Features
- Easy-to-navigate Code of Conduct and sport-specific sections
- Admin-editable content sections without touching code
- “Next games” chips with RSVP support
- PIN-based RSVP flow to avoid account creation while keeping basic access control
